# 👋 מדריך לתורמים (Contributing)

תודה על התעניינותך בתרומה ל-**X-Fix**! אנו מברכים כל סוג של תרומה - החל מתיקון באגים, הוספת פתרונות למאגר, ועד שיפורי קוד ועיצוב.

## 🛠️ איך מתחילים?

1.  **Fork:** צרו עותק של המאגר לחשבון שלכם.
2.  **Clone:** הורידו את הקוד למחשב המקומי.
3.  **Branch:** צרו ענף חדש עבור השינוי שלכם (`git checkout -b feature/amazing-feature`).
4.  **Code:** בצעו את השינויים. אנא הקפידו על קוד נקי וברור.
5.  **Commit:** בצעו שמירה עם הודעה ברורה (`git commit -m 'Add some amazing feature'`).
6.  **Push:** דחפו את השינויים ל-Fork שלכם.
7.  **Pull Request:** פתחו בקשה למיזוג השינויים למאגר הראשי שלנו.

## 📝 הוספת פתרון חדש למאגר

רוצים להוסיף פתרון סטטי? מעולה!
1. פתחו את הקובץ `src/data/mockSolutions.ts`.
2. הוסיפו אובייקט חדש למערך `MOCK_SOLUTIONS` לפי המבנה הקיים.
3. וודאו שהפתרון בדוק ומאומת.

## 🎨 סטנדרטים

*   אנו משתמשים ב-**TypeScript**. נא לא להשתמש ב-`any` אם אפשר להימנע מכך.
*   עיצוב מתבצע באמצעות **Tailwind CSS**.
*   הקפידו על כתיבת הערות (Comments) במקומות מורכבים.

תודה שאתם עוזרים לנו לתקן את העולם! 🌍
